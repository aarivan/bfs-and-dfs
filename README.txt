1) Extract the folder named Search_Romania
2) Copy and paste the file Search_Romania.java, Node_Class.java, cities.txt and adjacent.txt into the same folder
3) In the terminal, set the directory to the folder where the files are copied to
4) Compile using javac -Xlint:deprecation Search_Romania.java
5) To run, use java filename search_type start_node destination_node 

Where,
search_type is (dfs/bfs)
start_node is (Name of the start city)
destination_node is (Name of the destination city)
